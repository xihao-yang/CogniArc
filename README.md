# English

## Overview

**CogniArc** is an AI-native personal research blog and knowledge space.

It combines:

* Technical blogging
* Research documentation
* Learning notes
* An AI assistant that helps visitors explore the website

Instead of navigating only through menus, visitors can ask the AI guide questions such as:

> Where should I start reading?

The assistant can suggest relevant pages based on the site's content.

CogniArc aims to become a **long-term research notebook for AI systems, software engineering, and developer tools.**
